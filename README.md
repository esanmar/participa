## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## Localización y datos

- La interfaz sólo está disponible en **castellano y euskera**.
- Las incidencias son **puramente locales en memoria del navegador**; no hay base de datos ni persistencia en el servidor. Cada recarga reinicia los ejemplos iniciales.
